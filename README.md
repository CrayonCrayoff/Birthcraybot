Birthcraybot is a simple Discord bot that allows users to register their birthdays and shouts them out when that day comes! 

If you'd like to invite this bot to your server, click this button: [![Invite Bot](https://img.shields.io/badge/Invite-Bot-blue)](https://discord.com/oauth2/authorize?client_id=1434266532145533040&permissions=268618752&integration_type=0&scope=applications.commands+bot)

For help on setting it up, use the /help-user and /help-admin commands.
